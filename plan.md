# Plan de Implementación: Examen Formal con Carga Dinámica desde JSON

## Objetivo Principal

Transformar el examen actual en un **examen formal** que cargue preguntas dinámicamente desde `preguntas.json`, con sistema de contraseña para docente, generación de PDF y sin feedback visual durante el examen.

## Resumen de Requerimientos

### ✅ Durante el Examen:
- Carga automática desde JSON
- Sin feedback visual (no muestra correcto/incorrecto)
- Permite cambiar respuestas
- Estadísticas: Contestadas / Pendientes

### ✅ Al Finalizar:
- Botón "Enviar Examen"
- Modal de contraseña para docente
- Resultados detallados (qué acertó/falló)
- Descargar PDF con resultados
